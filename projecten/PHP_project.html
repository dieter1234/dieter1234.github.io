Over dit project

PHPproject

In semester 1 van het 2de jaar hadden we een opdracht om door heen het semester een php pagina te maken die ondersteuning had voor accounts,webshop, etc. Doorheen het semester ging het van simpele onepager naar een complexe met database gemaakte php pagina. Dit was geen gemakkelijk project voor mij omdat ik last had met alle variabelen te onthouden. Maar omdat ik de nodige vragen kan stellen aan de docent is het mij toch gelukt het af te krijgen. Deze website is gemaakt met behulp van php storm, laravel en homestead. php storm is een tool gemaakt door jetbrains. Laravel is een soort codering en homestead is een host platform.
