FiveM server

Eerst en vooral wat is een FiveM server? Een FiveM server is een privately hosted server voor het populaire spel gta5. Deze servers zijn gemaakt voor de spelers die met mods(extensiens voor de game zoals realistische auto's etc. zie foto's) willen spelen met hun vrienden. Als ze dit zouden willen doen op servers van Rockstar zouden ze geband worden. Daarom zijn FiveM servers gemaakt. Ik host deze server bij mij thuis op mijn raspberry py om samen met mijn vrienden te spelen. De OS op mijn raspberry py is redhat.
