Project 3

Project 3 was een project waarbij we iets moesten maken voor een opdrachtgever. In mijn geval was de opdrachtgever W. Kerkhofs die wouw dat er in elk lokaal gekeken kon worden hoe vol de lokalen effectief zaten. deze vraag was gekomen door het grote gebrek aan klaslokelen op de campus. Hierdoor zouden de lokalen beter ingedeeld kunnen worden. Onze oplossing was om met een camera om de x aantal minuten een photo te pakken van iedereen in het lokaal en dan gezichtsherkennings software te laten tellen hoeveel personen er effectief aanwezig zijn. Vervolgens werden de gegevens zonder de foto voor GDPR redenen naar een database verzonden. De gegevens van de database konden bekeken worden op een website die we hadden gemaakt voor de opdragt. waar kon gekeken worden per lokaal hoeveel studenten er aanwezig waren. De software, database en website werden gerund/host op een raspberry py. De gezichtsherkenning was een python script en de website werd op een apache server gehost zie foto's onderaan voor website.
